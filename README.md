#How to run

## Consolida datos de varios archivos en uno solo

```bash
$ ./join.sh todos.csv mesX.csv resultado.csv
```

## Cuenta de pagos y ultimo pago

```bash
# Genera archivo cuenta_pagos-<XXXXX>.csv
$ ./cuenta_pagos.sh resultados
```
